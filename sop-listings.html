Overview

The St. Andrews Listings page is a dedicated page showcasing all currently active properties at St. Andrews Country Club. This page is linked prominently from both the email and the main dashboard.

URL: https://edmundbogen.github.io/luxury-community-dashboard/st-andrews-listings.html

File: /luxury-community-dashboard/st-andrews-listings.html

Why a Separate Page?

St. Andrews Country Club is Edmund's primary focus community. Having a dedicated listings page allows subscribers to quickly see all available properties without navigating through the full dashboard. It also serves as a landing page for St. Andrews-specific marketing.

Page Structure

1. Header

Consistent branding with other pages:

  • Edmund Bogen Team logo
  • Title: "St. Andrews Country Club Listings"
  • Subtitle: "Boca Raton's Premier Golf Community"
  • Current listing count and price range

2. Summary Stats

Quick overview cards:

  • Total Active Listings
  • Price Range (Low - High)
  • Average Price
  • Average Days on Market

3. Listing Cards

Individual cards for each property with full details.

4. Contact CTA

Edmund's contact information and scheduling options.

Update Process

Step 1: Pull Current Inventory

Log into FlexMLS and search for:

  • Status: Active
  • Community: St. Andrews Country Club
  • Property Type: Single Family Residence

Step 2: Compare to Existing Page

Check for:

  • New listings: Add new cards
  • Sold/Pending: Remove those listings
  • Price changes: Update prices
  • Days on market: Update DOM for all listings

Step 3: Update Summary Stats

  • Total count
  • Price range (min - max)
  • Calculate new average price
  • Calculate new average DOM

Step 4: Generate FlexMLS Public Links

For each listing, generate a public link from FlexMLS that can be shared with non-MLS users.

Critical: Remove Sold Listings

Always verify that displayed listings are still active. Showing sold or pending properties creates a poor user experience and makes the data appear outdated. Check status before every update.

Sorting Order

Listings should be displayed in the following order:

  1. Price (High to Low) - Luxury positioning, showcase premium properties first
  2. Alternatively: Newest First - Show fresh inventory at top

Consistency

Use the same sorting order each week. If you change the sort order, be consistent going forward. The default recommendation is Price (High to Low) to emphasize the luxury market positioning.
