System Overview

This document provides complete instructions for producing the weekly "Luxury Communities Market Intelligence" email newsletter. The system consists of four interconnected components that work together to deliver market data and insights to our subscriber list.

Update Frequency: Components 1-3 are updated weekly. Component 4 (Featured Article) is created as needed for new content.

Weekly Production Workflow

Day 1: Data Collection

  • Pull MLS data for all 21 communities (active, sold, calculate absorption)
  • Pull St. Andrews specific listings with full details
  • Note significant market changes from prior week

Day 2: Dashboard & Listings Update

  • Update index.html with new community data
  • Update st-andrews-listings.html with current inventory
  • Update all chart data
  • Test all links, push to GitHub

Day 3: Article Creation (If Applicable)

  • Research topic and gather data
  • Write content following template structure
  • Create hero graphic
  • Build HTML page, push to GitHub

Day 4: Email Assembly

  • Copy email template, update date stamp
  • Update all statistics and community data
  • Update featured article section
  • Update featured listings (3 properties)
  • Preview and test all links

Day 5: Distribution

  • Send test email to Edmund for approval
  • Schedule or send to full list
  • Monitor open rates and clicks
