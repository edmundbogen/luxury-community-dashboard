Overview

The Luxury Communities Market Intelligence Dashboard is an interactive web page that displays comprehensive market data for all 21 tracked luxury communities. It provides visualizations, comparisons, and detailed statistics that give readers a deeper dive than the email summary.

URL: https://edmundbogen.github.io/luxury-community-dashboard/

File: /luxury-community-dashboard/index.html

Data Update Process

Step 1: Gather Data

Pull the following from MLS for each of the 21 communities:

  • Active listings count
  • Sold YTD count
  • Calculate absorption: Sold / (Sold + Active) × 100

Step 2: Update HTML

Open index.html and update:

  • Date stamp in header
  • Executive summary card values
  • Community table data
  • Chart data arrays (if using JavaScript charts)

Step 3: Update St. Andrews Listings

If any listings have changed, update the listing cards with:

  • Current active properties
  • Updated days on market
  • Any price changes
  • Remove sold properties

Tip: Efficiency

The dashboard and email share much of the same data. Update the dashboard first, then copy the relevant numbers to the email template. This ensures consistency.

Publishing Updates

GitHub Pages Workflow

  1. Make changes to local files
  2. Test locally by opening in browser
  3. Commit changes: git add . && git commit -m "Update market data [date]"
  4. Push to GitHub: git push
  5. Changes go live automatically (may take 1-2 minutes)

Important

Always verify the live site after pushing changes. GitHub Pages can occasionally cache old versions. If changes don't appear, try a hard refresh (Ctrl+Shift+R or Cmd+Shift+R).
